# Design System: Portfolio 2026

## Colors
- **Surface**: `#0e0e0e` (Base background)
- **Surface Container Low**: `#131313` (Section backgrounds)
- **Surface Container Highest**: `#262626` (Active/Hover states)
- **Primary**: `#c1fffe` (Cyan glow)
- **Secondary**: `#d575ff` (Purple glow)
- **Secondary Dim**: `#b90afc`
- **Text Primary**: `#ffffff` (On Background)
- **Text Secondary**: `#adaaaa` (On Surface Variant)
- **Border/Outline**: `#494847` (Outline Variant)

## Typography
- **Display/Headline**: `Inter` (Bold, -2% tracking)
- **Body**: `Manrope` (Regular/Medium)
- **Monospace/Labels**: `Space Grotesk`

## Motion (Framer Motion)
- **Easings**: `[0.2, 0.0, 0.0, 1.0]` (Weighted feel)
- **Scroll Reveals**: Fade + Slide Up
- **Parallax**: Subtle translateY on scroll
- **Tilt**: 3D hover effect on hero image

## UI Principles
- **No Borders**: Use background shifts (`#131313` on `#0e0e0e`) or whitespace.
- **Glassmorphism**: 40-60% opacity with 20px blur for floating elements.
- **Grain**: 3% opacity grain overlay for texture.
- **Cinematic**: Generous whitespace (80px+), high contrast.
